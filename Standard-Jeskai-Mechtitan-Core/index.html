Before We Start

在《神河霓朝記》中，可以把神器生物或載具合體成超級機器人的機體泰坦核心，風味滿點的設計正中機器人動畫世代的紅心。無獨有偶的，《兄弟之戰》推出史上第一張融合黑框旅法鵬洛客克撒，同樣勾動骨灰玩家情懷。這兩個組合在標準要怎麼玩才玩得動呢？就讓我們來合體一下吧！

揚升構念：主題分析

首先我們來看看這兩個組合的前置條件。

機體泰坦核心

{五}，放逐機體泰坦核心和四個由你操控的其他神器生物和／或載具：派出傳奇衍生神器生物機體泰坦，其為10/10，具飛行、警戒、踐踏、繫命與敏捷異能且是五色的組構體。當該衍生物離開戰場時，將所有以機體泰坦核心放逐的牌分別在其擁有者之操控下橫置移回戰場，但機體泰坦核心除外。
搭載2

護國勳爵克撒

你施放的神器、瞬間與巫術咒語減少{一}來施放。
{七}：如果由你同時擁有並操控護國勳爵克撒和一個名稱為強能石與弱能石的神器，則放逐它們，然後將它們融合為鵬洛客克撒。只能於巫術時機起動。

強能石與弱能石

當強能石與弱能石進戰場時，選擇一項～
‧抽兩張牌。
‧目標生物得-5/-5直到回合結束。{橫置}：加{無}{無}。此魔法力不能用來施放非神器咒語。
（與護國勳爵克撒融合。）

首先我們要進行機體泰坦核心的合體需要另外四個神器生物/載具，要累積到這個數量基本上有兩條路線：

  1. 不易被去除的載具。
  2. 容易展開的神器生物衍生物。

如果幫派劫庫械沒被禁的話載具或許是個不錯的路線，現階段誘因只剩駭流機體說實在有些不足，因此我們朝衍生物的方向構組，這個任務鵬洛客克撒也能做到。然而如果想要克撒進行有效率的合體需要大把魔力，剛好機體泰坦核心既是神器主題也同樣需要魔力合體，那麼魔力石路線就是合理的方向。

整修充能：套牌構築

既然走克撒魔力石路線，我們需要快速擴張魔法力拍出強能石與弱能石，要達到這個目的第一首選是索藍蜘蛛。2/4延勢帶一顆魔力石是堅實的三費中堅生物，又有啟動式異能可以消耗過剩魔力，當然塞滿四張。然而只靠四張蜘蛛加速不夠穩當，雖然護國勳爵克撒也能分攤加速工作，但他的脖子不夠硬、又是不好回收的致勝手段，這樣隨便施放吃到去除就虧了。所以我們從魔力石相關單卡中挑選了兼顧去除功能的掘古爆破，加速同時妨礙對手展開，可以直接往對手臉上招呼這點是筆者中意的地方。

搞定套牌骨幹後先來決定大招，在考量機體泰坦核心需求的前提下，能派出複製品衍生物的掠束大隊自然是首選，可視情況試作施放直接合體；另外這副套牌缺乏對大型生物及神器結界的處理手段，這點我們需要夷城械來做彌補；巨型原身械的複製效應與掠束大隊巧妙互動，正常施放並選擇複製掠束大隊時不僅本體是7/7，加上符合觸發進場異能的條件所以會派出複製品衍生物，而且派出的衍生物也是7/7，高達21點的敏捷踐踏打點可以直接終結遊戲，也可視情況複製強能石與弱能石賺取牌張資源。

接下來就是低費生物與去除。雖然金鑄振翼龍、熱狂馭械師看似是不錯的二費選擇，但以現代二費生物動輒3/2的體格來說根本擋不住，更別提有熊野傳記加持的寺院迅矛僧，幾經測試後選用自帶繫命的強顎裁斷僧，一來防守時能換掉頑強敗犬、血稅收集官並回復生命，搭配索藍蜘蛛、掘古爆破主動進攻也能給對手造成不小的壓力，同時給自己帶來更多血量上的餘裕。另外作為幫派劫庫械的替代，我們使用實界晶片來阻擋前期三攻生物，中後期配帶後更可從牌庫頂持續賺取資源。雙箭神射手則以魂力作為兩費去除曲線，前期可以繞過瑟班守護者莎利雅的干擾擊殺2防生物，四費正拍2 / 3延勢附帶一發電震射掉小生物、鵬洛客，甚至有機會收頭。作為主題的機體泰坦核心雖然本體不怎麼樣，2/4的身材還是可以頂住不少中小型生物。


剩下的五個位置我們填入絲金大師莎希莉、崇史學徒羅蘭與進軍卡拉德許。絲金大師莎希莉的技能組完美契合神器套牌的需求，派出的振翼機衍生物也能作為機體泰坦核心的合體材料；崇史學徒羅蘭負責回收神器，不僅和實界晶片、護國勳爵克撒互動良好，還可以和巨型原身械玩出神器回收循環組合技；進軍卡拉德許的進場效果雖然不怎麼樣，轉化後的金鱗旗艦乙太翼號在這副套牌卻是鬼神般的存在，隨隨便便就有6以上的神器飛行打點，有時候光靠它就能輕鬆擊敗對手。

結語

如同影片所示，這副套牌的運作方式是前期一邊用強顎裁斷僧向對手施壓一邊開礦種魔力石，掘古爆破、索藍蜘蛛、強能石與弱能石阻撓對手的展開兼加速，等到湊到元件即可請對手吃一套合體技，打到殘血後通常再補一發掘古爆破或雙箭神射手就能結束遊戲。覺得這個套路很酷、喜愛超級機器人的讀者可以組來玩看看，強度或許見仁見智，樂趣肯定不會少。
