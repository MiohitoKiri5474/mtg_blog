十會碎盟：標準至尊再起

Before We Start

《卡洛夫莊園謀殺案》引領我們四度重返拉尼卡，也帶來一系列的強力雙色單卡，加上標準牌池內的五色領土基底，為《終戰迴響》的至尊尼米捷提供了登上舞台的絕佳背景環境，這就來構組看看吧。

單卡解析


至尊尼米捷

飛行，反單色辟邪
你墳墓場中每張正好兩色的瞬間與巫術牌均具有再起異能。（你可以從你的墳墓場施放該牌，但必須支付其所需費用並額外棄一張牌。然後將它放逐。）

作為拉尼卡的現世十會盟與伊捷聯盟的元祖，至尊尼米捷為墳場中所有公會色組的瞬間巫術牌加上再起異能，不僅沒有翻湧神信徒列爾的咒語不能被反擊debuff，本體還是5/5飛行反單色辟邪的強力終結者，在當前的賽場環境中幾乎迴避了跳船跟掃場以外的所有去除手段。套牌構築如下：

套牌列表

為了與至尊尼米捷的搭配，我們在新系列獲得了潔思凱色組三連星－閃電螺旋、終結謊言與厄時爆炸，同時顧到短解、反擊、補血、抽牌與掃場。閃電螺旋牌子老效果好，不僅可以提高前期生存率，後期還可利用再起給予對手致命一擊。終結謊言是新世代的魔力流失，額外附帶放逐在面對有墳場利用能力的對手時有機會發揮奇效。厄時爆炸除了如同異能字面上可以當作掃場使用外，也能選擇不棄牌當作卜卦使用，對到沒有掃場需求的對局也不會成為死牌。

在現行標準要走五色，領土路線絕對是最佳選擇，地脈束縛與獸群遷徙分別補強去除與調色補血的能力，但是進軍贊迪卡不是這副控制套牌所需要的，所以筆者替換成可以和至尊尼米捷做搭配的十會碎盟案，前期調色不用擔心卡綠，後期可讓尼米捷直接升格成連擊警戒的鬼神。

以上的搭配足夠嗎？就如同前面所提到的，至尊尼米捷雖然不吃單色點殺，面對掃場或許還能靠終結謊言反擊處理，遇上跳船就相當脆弱。所以在五費終結者的位置上筆者做了一些混搭，加入碑出告與開璃、朱高與歐祝泰兩個龍族好夥伴，碑出告與開璃只要進場就能靠腦力激盪補充一張手牌，對手要用黑色去除處理還得賭賭看你的牌庫頂不是獸群遷徙之類的大招。至於朱高與歐祝泰請參見前文，另外再搭配一張進軍拉尼卡
作為額外的去除與資源機。


最後再為以上架構做個輔助搭配－推出/拉回與緊急剖檢。推出/拉回主要使用拉回那面去回收進墳場的龍，可以跟開璃與歐祝泰做搭配，前者可以打完一拳後犧牲再賺一次腦力激盪與牌庫頂的招，後者可以讓龍回手迴避犧牲異能，只是要注意推出/拉回是三色牌，無法獲得再起能力。緊急剖檢是一換多的強力瞬間去除，但需要大費用的蒐證搭配，這點可以透過獸群遷徙或厄時爆炸來滿足條件。

實戰演示

兩支演示影片分別就A計畫至尊尼米捷與B計畫歐祝泰／開璃的實戰狀況進行展演，第一支影片中至尊尼米捷憑藉無恥的反單色辟邪與再起帶來的牌差優勢直接控死對手，在第二支影片中雖然意外吃到了熔毀崩塌沒賺到再起，卻也偵結了十會碎盟案，讓歐祝泰成為超強敏捷打點兼賺牌機器，最後的開璃如果對手使用一般去除處理至少得付出五點生命代價（如果他生命還夠的話），去除時機不對的話還得面對獸群遷徙帶來的五支3/3野獸，同時手上還捏著康應對意外狀況，可說是勝券在握。

結論

《卡洛夫莊園謀殺案》恰到好處地為至尊尼米捷做了球，在領土穩定的五色基底上使得尼米捷控制成為可能。雖然因為大量躺地的配置讓這副控制套牌第二回合順利做招的機會比起正常三色控制來得低，對上紅白召集或紅快攻討不太到更多好處，但比起傳統的領土套牌，大量的干擾咒語也意味著較少的空隙，對於打膩加速的玩家來說會是相當值得嘗試的一個變化版本。
